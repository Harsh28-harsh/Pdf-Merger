# Pdf-Merger

A simple and efficient web-based tool to merge multiple PDF files into a single document.

## 🚀 Features
- Merge multiple PDFs easily
- Fast and lightweight processing
- Simple and user-friendly interface
- Works directly in browser / Node.js (if applicable)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Node.js (if used)

## 📌 How to Use
1. Upload your PDF files
2. Arrange them in order
3. Click on Merge
4. Download your combined PDF

## 💡 Purpose
Built to make PDF merging quick, simple, and accessible without heavy software.
